# เริ่มต้นใช้งาน

คู่มือนี้จะแนะนำวิธีการเริ่มต้นใช้งาน bizdocgen ตั้งแต่ขั้นตอนแรกจนถึงการสร้างเอกสารแรกของคุณ

## ข้อกำหนดเบื้องต้น

ก่อนเริ่มใช้งาน bizdocgen คุณต้องมี:

- บัญชี Grist (สมัครฟรีได้ที่ [getgrist.com](https://www.getgrist.com/))
- เบราว์เซอร์ที่รองรับ JavaScript (Chrome, Firefox, Safari, Edge)

## ขั้นตอนพื้นฐานหลังติดตั้ง

หลังจากติดตั้งเสร็จแล้ว คุณสามารถเริ่มใช้งานได้ทันที:

### 1. สร้างเอกสารใหม่

1. เปิดตาราง **Documents** ใน Grist
2. เพิ่มแถวใหม่เพื่อสร้างเอกสาร
3. เลือกประเภทเอกสาร (Quotation/Invoice/Receipt)
4. กรอกข้อมูลลูกค้า ผู้ให้บริการ และรายการสินค้า

### 2. ดูตัวอย่างเอกสาร

Widget จะแสดงตัวอย่างเอกสารแบบ Real-time เมื่อคุณแก้ไขข้อมูลในตาราง

### 3. พิมพ์หรือบันทึกเป็น PDF

1. คลิกปุ่ม **🖨️ พิมพ์เอกสาร**
2. ใน dialog พิมพ์ เลือก **Save as PDF**
3. ปรับการตั้งค่าตามต้องการแล้วบันทึก

## โครงสร้างข้อมูลพื้นฐาน

bizdocgen ใช้ข้อมูลจากตารางต่างๆ ใน Grist:

| ตาราง | รายละเอียด |
|-------|-----------|
| **Documents** | ข้อมูลเอกสารหลัก (เลขที่ วันที่ ประเภท) |
| **Clients** | ข้อมูลลูกค้า (ชื่อ ที่อยู่ เลขประจำตัวผู้เสียภาษี) |
| **Providers** | ข้อมูลผู้ให้บริการ (ชื่อ ที่อยู่ ข้อมูลติดต่อ) |
| **Items** | รายการสินค้า/บริการ (รายละเอียด จำนวน ราคา) |
| **Payment Methods** | วิธีการชำระเงิน (ธนาคาร บัญชี PromptPay) |

## ตัวอย่างการใช้งาน

### สร้างใบเสนอราคา

```
1. สร้างเอกสารใหม่ในตาราง Documents
2. เลือก Document_Type = "Quotation"
3. เลือกลูกค้าจากตาราง Clients
4. เพิ่มรายการสินค้าในตาราง Items
5. กำหนดเงื่อนไขการชำระเงิน (Credit_Term)
6. ดูตัวอย่างและพิมพ์
```

### สร้างใบแจ้งหนี้พร้อม PromptPay

```
1. สร้างเอกสารใหม่
2. เลือก Document_Type = "Invoice"
3. เลือกวิธีการชำระเงินที่มี PromptPay ID
4. ระบบจะสร้าง QR Code อัตโนมัติ
5. ลูกค้าสามารถสแกนจ่ายได้ทันที
```

### สร้างใบเสร็จพร้อมภาษีหัก ณ ที่จ่าย

```
1. สร้างเอกสารใหม่
2. เลือก Document_Type = "Receipt"
3. ตั้งค่า Tax = -0.03 (สำหรับ WHT 3%)
4. ระบบจะคำนวณภาษีหัก ณ ที่จ่ายอัตโนมัติ
```
