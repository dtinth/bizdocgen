# ใช้งานบน Hosted Grist

วิธีที่ง่ายที่สุดในการเริ่มต้นใช้งาน bizdocgen คือการใช้งานบน Grist Cloud (Hosted Grist) ซึ่งเป็นบริการที่ทาง Grist ให้บริการ

## เริ่มต้นใช้งานทันที

### ขั้นตอนที่ 1: เปิดเทมเพลต

[คลิกที่นี่เพื่อเปิดเทมเพลต](https://bizdocgen.getgrist.com/cq6sb6WHRMre/bizdocgen-template){ .md-button .md-button--primary }

หรือเข้าไปที่ URL: `https://bizdocgen.getgrist.com/cq6sb6WHRMre/bizdocgen-template`

### ขั้นตอนที่ 2: ทดลองใช้งาน (Fiddle Mode)

เมื่อเปิดเทมเพลตครั้งแรก คุณจะอยู่ใน **Fiddle Mode**:

- สามารถแก้ไขข้อมูลได้ทันทีโดยไม่ต้องลงชื่อเข้าใช้
- การเปลี่ยนแปลงจะไม่ถูกบันทึก
- เหมาะสำหรับทดลองใช้งานก่อนตัดสินใจ

### ขั้นตอนที่ 3: บันทึกลงบัญชีของคุณ

หากต้องการบันทึกและใช้งานจริง:

1. คลิกปุ่ม **Use This Template** หรือ **Save Copy**
2. ลงชื่อเข้าใช้หรือสมัครบัญชี Grist (ฟรี)
3. เทมเพลตจะถูกคัดลอกไปยัง workspace ของคุณ

## โครงสร้างเทมเพลต

เทมเพลตมาพร้อมกับข้อมูลตัวอย่างที่ครบถ้วน:

| ตาราง | รายละเอียด |
|-------|-----------|
| **Documents** | เอกสารตัวอย่าง 3 ประเภท |
| **Clients** | ข้อมูลลูกค้าตัวอย่าง |
| **Providers** | ข้อมูลผู้ให้บริการตัวอย่าง |
| **Items** | รายการสินค้าตัวอย่าง |
| **Payment Methods** | วิธีการชำระเงินตัวอย่าง |

## การตั้งค่า Custom Widget

หากต้องการตั้งค่า Widget ด้วยตนเอง:

1. เปิดเอกสารในตาราง Documents
2. คลิกที่ปุ่ม **Add Widget** หรือแก้ไข widget ที่มีอยู่
3. เลือก **Custom Widget**
4. ตั้งค่า URL:
   - ใช้ URL จากการ build: `https://<your-domain>/`
   - หรือใช้ URL จาก GitHub Pages

### การตั้งค่า Access

ตรวจสอบให้แน่ใจว่า widget มีสิทธิ์เข้าถึงข้อมูล:

```
Widget Options → Access → Full document access
```

## ข้อจำกัดของ Free Tier

Grist Free Tier มีข้อจำกัดดังนี้:

- จำกัด 5,000 แถวต่อเอกสาร
- 5 ผู้ใช้ต่อเอกสาร
- API rate limits

สำหรับธุรกิจขนาดเล็ก ข้อจำกัดนี้เพียงพอสำหรับการใช้งานหลายปี

> [!TIP]
> หากต้องการข้อมูลเพิ่มเติม สามารถดูรายละเอียดที่ [Grist Pricing](https://www.getgrist.com/pricing/)

## การสำรองข้อมูล

คุณสามารถสำรองข้อมูลเอกสาร Grist ได้โดย:

1. เปิดเมนู **Document** → **Download**
2. เลือกรูปแบบ:
   - **.grist** - ไฟล์ Grist ที่สามารถนำเข้ากลับได้
   - **.csv** - สำหรับแต่ละตาราง
