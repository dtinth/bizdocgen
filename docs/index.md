# bizdocgen

bizdocgen คือวิดเจ็ตแบบกำหนดเองของ [Grist](https://support.getgrist.com/widget-custom/) สำหรับสร้างเอกสารธุรกิจภาษาไทยจากข้อมูลในตาราง เช่น ใบเสนอราคา ใบแจ้งหนี้ และใบเสร็จรับเงิน

## สิ่งที่ทำได้

- สร้างเอกสาร 3 ประเภท: **ใบเสนอราคา**, **ใบแจ้งหนี้**, **ใบเสร็จรับเงิน**
- คำนวณภาษีมูลค่าเพิ่ม 7% และภาษีหัก ณ ที่จ่าย
- แสดงข้อมูลการชำระเงินผ่านบัญชีธนาคารและ **PromptPay QR Code**
- พิมพ์เอกสารได้ทันทีจากหน้าเว็บ
- บันทึก **Custom CSS** ไว้ในตัวเลือกของวิดเจ็ต เพื่อปรับหน้าตาเอกสาร
- ใช้งานได้ทั้งใน Grist และโหมด standalone สำหรับพัฒนา/ทดสอบ

!!! tip
    ถ้าต้องการลองใช้งานเร็วที่สุด ให้เริ่มจากเทมเพลต Grist ที่โฮสต์ไว้ แล้วค่อยคัดลอกไปยัง workspace ของตัวเอง

## ภาพรวมการทำงาน

1. Grist ส่งข้อมูลของระเบียนปัจจุบันเข้ามาที่วิดเจ็ต
2. แอปตรวจสอบข้อมูลด้วย schema ของ Zod
3. แอปแปลงข้อมูลเป็น view model สำหรับแสดงผล
4. Vue components หลายส่วนประกอบกันเป็นเอกสารที่พร้อมพิมพ์
5. ถ้ามีข้อมูล PromptPay ระบบจะสร้าง QR Code ให้โดยอัตโนมัติ
