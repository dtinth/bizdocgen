# bizdocgen

**เครื่องมือสร้างเอกสารธุรกิจอัตโนมัติ ผ่าน [Grist](https://www.getgrist.com/)**

![ตัวอย่างเอกสาร](example.png)

## bizdocgen คืออะไร?

bizdocgen เป็น Custom Widget สำหรับ [Grist](https://www.getgrist.com/) ที่ช่วยสร้างเอกสารธุรกิจจากข้อมูลในสเปรดชีตโดยอัตโนมัติ รองรับเอกสาร 3 ประเภท:

- **ใบเสนอราคา** (Quotation)
- **ใบแจ้งหนี้** (Invoice)
- **ใบเสร็จรับเงิน** (Receipt)

เหมาะสำหรับฟรีแลนซ์ ที่ปรึกษา และเจ้าของธุรกิจขนาดเล็กที่ต้องการออกเอกสารทางธุรกิจอย่างเป็นระบบ

## ความสามารถหลัก

- :material-file-document: สร้างเอกสาร 3 ประเภท — ใบเสนอราคา ใบแจ้งหนี้ ใบเสร็จรับเงิน
- :material-calculator: คำนวณภาษีอัตโนมัติ — VAT 7% และภาษีหัก ณ ที่จ่าย
- :material-qrcode: สร้าง QR Code พร้อมเพย์ — สำหรับรับชำระเงิน
- :material-printer: พิมพ์เอกสารขนาด A4 — จัดหน้าพร้อมพิมพ์ทันที
- :material-palette: ปรับแต่ง CSS ได้ — เปลี่ยนฟอนต์ สี และรูปแบบตามต้องการ
- :material-language-markdown: รองรับ Markdown — ในรายละเอียดสินค้าและหมายเหตุ

## Grist คืออะไร?

[Grist](https://www.getgrist.com/) เป็นแพลตฟอร์มสเปรดชีตสมัยใหม่ที่รองรับ Custom Widget ซึ่งช่วยให้สามารถแสดงผลข้อมูลในสเปรดชีตในรูปแบบต่าง ๆ ได้ bizdocgen ใช้ความสามารถนี้ในการแปลงข้อมูลในสเปรดชีตให้เป็นเอกสารธุรกิจที่พร้อมพิมพ์
