# คุณสมบัติ

bizdocgen มาพร้อมกับคุณสมบัติที่ครบถ้วนสำหรับการออกเอกสารทางธุรกิจในประเทศไทย

## เอกสารที่รองรับ

<div class="grid cards" markdown>

-   :material-file-document-outline:{ .lg .middle } __ใบเสนอราคา (Quotation)__

    ---

    เอกสารเสนอราคาสินค้าหรือบริการให้กับลูกค้า แสดงรายละเอียด ราคา และเงื่อนไขการชำระเงิน

-   :material-file-document-edit:{ .lg .middle } __ใบแจ้งหนี้ (Invoice)__

    ---

    เอกสารแจ้งหนี้เรียกเก็บเงิน รองรับการแสดง QR Code สำหรับชำระเงิน

-   :material-file-document-check:{ .lg .middle } __ใบเสร็จรับเงิน (Receipt)__

    ---

    เอกสารยืนยันการรับชำระเงิน แสดงรายละเอียดการชำระและยอดรวมสุทธิ

</div>

## คุณสมบัติทั้งหมด

### การจัดการเอกสาร

| คุณสมบัติ | รายละเอียด |
|-----------|-----------|
| **เลขที่เอกสารอัตโนมัติ** | กำหนดรูปแบบเลขที่เอกสารได้เอง |
| **วันที่เอกสาร** | รองรับรูปแบบวันที่ไทย |
| **อ้างอิงเอกสาร** | เชื่อมโยงใบเสนอราคา-ใบแจ้งหนี้-ใบเสร็จ |
| **หมายเหตุ** | รองรับ Markdown formatting |

### การคำนวณภาษี

| คุณสมบัติ | รายละเอียด |
|-----------|-----------|
| **VAT 7%** | คำนวณภาษีมูลค่าเพิ่มอัตโนมัติ |
| **ภาษีหัก ณ ที่จ่าย** | รองรับ WHT 1%, 2%, 3%, 5% |
| **การแสดงผลภาษี** | แยกแสดงรายการภาษีอย่างชัดเจน |

### การชำระเงิน

| คุณสมบัติ | รายละเอียด |
|-----------|-----------|
| **PromptPay QR** | สร้าง QR Code อัตโนมัติจากหมายเลข PromptPay |
| **ข้อมูลบัญชีธนาคาร** | แสดงชื่อบัญชี เลขที่บัญชี ธนาคาร และสาขา |
| **เงื่อนไขการชำระ** | กำหนดเครดิตเทอมได้ (เช่น 30 วัน) |

### การปรับแต่ง

| คุณสมบัติ | รายละเอียด |
|-----------|-----------|
| **Custom CSS** | ปรับแต่งหน้าตาเอกสารด้วย CSS |
| **Font ไทย** | ใช้ฟอนต์ Sarabun มาตรฐานไทย |
| **การพิมพ์** | รองรับการพิมพ์และบันทึก PDF |

## การใช้งาน

### สลับระหว่างประเภทเอกสาร

เปลี่ยนประเภทเอกสารได้ทันทีโดยแก้ไขฟิลด์ `Document_Type`:

```
Quotation  → ใบเสนอราคา
Invoice    → ใบแจ้งหนี้
Receipt    → ใบเสร็จรับเงิน
```

### การคัดลอกข้อมูล JSON

คลิกปุ่ม **📋 คัดลอก JSON** เพื่อคัดลอกข้อมูลเอกสารในรูปแบบ JSON สำหรับ:
- การ debug
- การ integrate กับระบบอื่น
- การส่งข้อมูลให้นักพัฒนา

### การพิมพ์เอกสาร

1. คลิกปุ่ม **🖨️ พิมพ์เอกสาร**
2. ระบบจะเปิด dialog พิมพ์ของเบราว์เซอร์
3. เลือก **Save as PDF** หรือปริ้นเตอร์

> [!NOTE]
> ปุ่มและส่วนตั้งค่าจะถูกซ่อนอัตโนมัติเมื่อพิมพ์
