# การจัดการภาษี

bizdocgen รองรับการคำนวณและแสดงผลภาษีตามมาตรฐานไทย

## ประเภทภาษีที่รองรับ

### VAT 7% (ภาษีมูลค่าเพิ่ม)

คำนวณ VAT 7% อัตโนมัติเมื่อตั้งค่า `Tax = 0.07`

```
ยอดรวม:      10,000.00 บาท
VAT 7%:         700.00 บาท
ยอดสุทธิ:    10,700.00 บาท
```

### ภาษีหัก ณ ที่จ่าย (Withholding Tax)

รองรับการหักภาษี ณ ที่จ่ายโดยใช้ค่าติดลบ:

| อัตราภาษี | ค่าที่ตั้ง | ผลลัพธ์ |
|-----------|------------|---------|
| WHT 1% | -0.01 | หัก 1% |
| WHT 2% | -0.02 | หัก 2% |
| WHT 3% | -0.03 | หัก 3% |
| WHT 5% | -0.05 | หัก 5% |

```
ยอดรวม:             10,000.00 บาท
ภาษีหัก ณ ที่จ่าย 3%:  -300.00 บาท
ยอดสุทธิ:            9,700.00 บาท
```

### ไม่มีภาษี

ตั้งค่า `Tax = 0` หรือ `null` สำหรับเอกสารที่ไม่มีภาษี

```
ยอดรวม:      10,000.00 บาท
ยอดสุทธิ:    10,000.00 บาท
```

## การตั้งค่าใน Grist

### ขั้นตอน

1. เปิดตาราง **Documents**
2. หาคอลัมน์ **Tax**
3. ใส่ค่าตามประเภทภาษีที่ต้องการ:
   - `0.07` = VAT 7%
   - `-0.03` = WHT 3%
   - `0` = ไม่มีภาษี

### ตัวอย่างข้อมูล

| Number | Document_Type | Tax | ผลลัพธ์ |
|--------|---------------|-----|---------|
| QT-001 | Quotation | 0.07 | VAT 7% |
| INV-001 | Invoice | -0.03 | WHT 3% |
| RC-001 | Receipt | 0 | ไม่มีภาษี |

## การแสดงผลภาษี

### Label ที่แสดง

ระบบจะแสดง label ภาษีอัตโนมัติตามประเภท:

- **VAT 7%** → "VAT 7%"
- **WHT** → "ภาษีหัก ณ ที่จ่าย X%"
- **ไม่มีภาษี** → ไม่แสดงรายการภาษี

### การคำนวณ

```typescript
// VAT 7%
const vatAmount = subtotal * 0.07
const total = subtotal + vatAmount

// WHT 3%
const whtAmount = subtotal * -0.03
const total = subtotal + whtAmount  // หักลบ

// No tax
const total = subtotal
```

## กรณีพิเศษ

### VAT + WHT พร้อมกัน

หากต้องการ VAT และ WHT พร้อมกัน ให้คำนวณในรายการสินค้าแทน:

1. ตั้งค่า `Tax = 0` ในเอกสาร
2. คำนวณ VAT และ WHT ในราคารายการ
3. ใส่รายละเอียดในหมายเหตุ

### การปัดเศษ

ระบบคำนวณโดยใช้ทศนิยม 2 ตำแหน่งตามมาตรฐานไทย:

```typescript
// ตัวอย่างการคำนวณ
const subtotal = 1000
const vat = (subtotal * 0.07).toFixed(2)  // "70.00"
const total = (subtotal + parseFloat(vat)).toFixed(2)  // "1070.00"
```

## การแก้ไขปัญหา

### ภาษีไม่แสดง

ตรวจสอบ:
- ค่า Tax ต้องเป็นตัวเลข (number) ไม่ใช่ข้อความ
- ค่าต้องอยู่ระหว่าง -1 ถึง 1 (เช่น 0.07 ไม่ใช่ 7)

### ยอดรวมไม่ถูกต้อง

ตรวจสอบ:
- ราคารายการ (Unit_Price) × จำนวน (Quantity) = ราคารวม (Total)
- ค่า Tax มีเครื่องหมายถูกต้อง (บวกสำหรับ VAT ลบสำหรับ WHT)

### ต้องการซ่อนรายการภาษี

ตั้งค่า `Tax = 0` เพื่อซ่อนรายการภาษีทั้งหมด

## CSS Classes

สามารถปรับแต่งการแสดงผลภาษีด้วย CSS:

```css
/* ส่วนแสดงภาษี */
.tax-summary {
  margin-top: 1rem;
}

/* รายการภาษี */
.tax-summary__row {
  display: flex;
  justify-content: space-between;
}

/* Label ภาษี */
.tax-summary__label {
  font-weight: 500;
}

/* จำนวนเงินภาษี */
.tax-summary__amount {
  font-family: var(--font-family-mono);
}

/* WHT - แสดงเป็นสีแดง */
.tax-summary__row:has(.tax-summary__label[data-type="wht"]) {
  color: #dc2626;
}
```
