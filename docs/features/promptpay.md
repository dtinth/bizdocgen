# PromptPay QR Code

bizdocgen รองรับการสร้าง QR Code สำหรับรับชำระเงินผ่าน PromptPay อัตโนมัติ

## คุณสมบัติ

- สร้าง QR Code ตามมาตรฐาน PromptPay
- รองรับการระบุจำนวนเงินอัตโนมัติ
- แสดงข้อมูลบัญชีธนาคารควบคู่
- รองรับทั้งหมายเลขโทรศัพท์และเลขบัตรประชาชน

## การตั้งค่า

### ขั้นตอนที่ 1: สร้างวิธีการชำระเงิน

1. เปิดตาราง **Payment Methods**
2. เพิ่มแถวใหม่
3. กรอกข้อมูล:
   - **Name** - ชื่อวิธีการชำระ (เช่น "บัญชีธนาคารกสิกร")
   - **PromptPay** - หมายเลข PromptPay
   - **Bank** - ชื่อธนาคาร
   - **Branch** - สาขา
   - **Account Holder** - ชื่อเจ้าของบัญชี
   - **Account Number** - เลขที่บัญชี

### ขั้นตอนที่ 2: เชื่อมโยงกับเอกสาร

1. เปิดตาราง **Documents**
2. เลือกเอกสารที่ต้องการ
3. ในคอลัมน์ **Payment Method** เลือกวิธีการชำระเงินที่สร้างไว้

### ขั้นตอนที่ 3: ตรวจสอบ

Widget จะแสดง:
- ข้อมูลบัญชีธนาคาร
- QR Code สำหรับสแกนจ่าย

## รูปแบบ PromptPay ID

PromptPay รองรับ 2 รูปแบบ:

| รูปแบบ | ตัวอย่าง | ความยาว |
|--------|---------|---------|
| หมายเลขโทรศัพท์ | 0812345678 | 10 หลัก |
| เลขบัตรประชาชน | 1234567890123 | 13 หลัก |

> [!NOTE]
> ไม่ต้องใส่เครื่องหมายขีด (-) หรือช่องว่าง ระบบจะจัดการให้อัตโนมัติ

## การทำงาน

### การสร้าง QR Code

เมื่อเอกสารมี Payment Method ที่ระบุ PromptPay:

1. ระบบคำนวณยอดรวมสุทธิ
2. สร้าง payload ตามมาตรฐาน PromptPay
3. สร้าง QR Code รูปแบบ PNG
4. แสดงในเอกสาร

### การสแกนจ่าย

ลูกค้าสามารถ:

1. เปิดแอพธนาคารบนมือถือ
2. เลือกสแกน QR Code
3. สแกน QR Code บนเอกสาร
4. ตรวจสอบจำนวนเงิน
5. ยืนยันการชำระเงิน

## การแก้ไขปัญหา

### QR Code ไม่แสดง

ตรวจสอบ:
- มีการเลือก Payment Method ในเอกสาร
- PromptPay ID ถูกต้อง (10 หรือ 13 หลัก)
- ไม่มีอักขระพิเศษใน PromptPay ID

### สแกนไม่ได้

ตรวจสอบ:
- QR Code ชัดเจน (ไม่เบลอ)
- แสงเพียงพอ
- แอพธนาคารรองรับ PromptPay

### จำนวนเงินไม่ถูกต้อง

ตรวจสอบ:
- รายการสินค้าคำนวณถูกต้อง
- การตั้งค่าภาษีถูกต้อง
- ยอดรวมสุทธิถูกต้อง

## ความปลอดภัย

- QR Code สร้างจากข้อมูลในเอกสารเท่านั้น
- ไม่มีการเก็บข้อมูลการเงินบน server
- การชำระเงินทำผ่านแอพธนาคารโดยตรง

## CSS Customization

ปรับแต่งการแสดงผล QR Code:

```css
/* ซ่อน QR Code ในเอกสารบางประเภท */
[data-document-type="quotation"] .qr-code-section {
  display: none;
}

/* ปรับขนาด QR Code */
.qr-code-section__image {
  width: 150px;
  height: 150px;
}

/* ย้ายตำแหน่ง QR Code */
.document__payment-section {
  flex-direction: column;
  align-items: center;
}
```
