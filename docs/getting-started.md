# เริ่มต้นใช้งาน

## วิธีที่ 1: ใช้งานผ่าน Hosted Grist (ง่ายที่สุด)

วิธีนี้เหมาะสำหรับผู้ที่ต้องการทดลองใช้งานทันที โดยไม่ต้องติดตั้งอะไรเพิ่มเติม

1. **เปิดเทมเพลต** — [คลิกที่นี่เพื่อเปิดเทมเพลต](https://bizdocgen.getgrist.com/cq6sb6WHRMre/bizdocgen-template) (ไม่ต้องสมัครสมาชิก)
2. **ทดลองแก้ไข** — สามารถแก้ไขข้อมูลได้เลย ระบบจะสร้างสำเนาชั่วคราว ([Fiddle Mode](https://support.getgrist.com/glossary/#fiddle-mode)) ให้ทดลองใช้
3. **บันทึกเป็นของตัวเอง** — กดปุ่ม **Use This Template** หรือ **Save Copy** เพื่อบันทึกไปยัง Workspace ของคุณ (ต้องสมัครสมาชิก Grist)

!!! tip "Grist ฟรีเพียงพอ"
    Grist มี [แผนฟรี](https://www.getgrist.com/pricing/) ที่รองรับ 5,000 แถวต่อเอกสาร ซึ่งเพียงพอสำหรับการใช้งานทั่วไป

## วิธีที่ 2: ใช้งานกับ Self-Hosted Grist

สำหรับผู้ที่ต้องการควบคุมข้อมูลเองทั้งหมด สามารถ [ติดตั้ง Grist บนเซิร์ฟเวอร์ของตัวเอง](https://support.getgrist.com/self-managed/) ได้

1. **ดาวน์โหลดเทมเพลต** — ดาวน์โหลดไฟล์ [template.grist](https://github.com/dtinth/bizdocgen/raw/main/template.grist) จาก Repository
2. **นำเข้า Grist** — Import ไฟล์เทมเพลตเข้า Grist ของคุณ

!!! tip "ข้อดีของ Self-Hosted"
    การ [ติดตั้ง Grist เอง](https://support.getgrist.com/self-managed/) ช่วยให้ข้อมูลอยู่ภายใต้การควบคุมของคุณ ไม่จำกัดจำนวนแถว และไม่มี API Rate Limit

## การใช้งานเบื้องต้น

### การกรอกข้อมูล

เมื่อเปิดเทมเพลตแล้ว จะเห็นตารางสเปรดชีตที่มีคอลัมน์ต่าง ๆ ดังนี้:

| คอลัมน์ | คำอธิบาย |
|---------|----------|
| `Document_Type` | ประเภทเอกสาร — Quotation, Invoice, หรือ Receipt |
| `Number` | เลขที่เอกสาร เช่น `INV-2025-001` |
| `Date` | วันที่ออกเอกสาร |
| `Provider` | ข้อมูลผู้ออกเอกสาร (ชื่อ ที่อยู่ เลขประจำตัวผู้เสียภาษี) |
| `Client` | ข้อมูลลูกค้า (ชื่อ ที่อยู่ เลขประจำตัวผู้เสียภาษี) |
| `Items` | รายการสินค้า/บริการ |
| `Tax` | อัตราภาษี (เช่น `0.07` สำหรับ VAT 7%) |
| `Payment_Method` | วิธีการชำระเงิน (บัญชีธนาคาร, พร้อมเพย์) |
| `Remarks` | หมายเหตุ (รองรับ Markdown) |

### การดูตัวอย่างเอกสาร

เมื่อเลือกแถวในสเปรดชีต Widget จะแสดงตัวอย่างเอกสารที่สร้างจากข้อมูลในแถวนั้นทันที

### การพิมพ์เอกสาร

กดปุ่ม **🖨️ พิมพ์** เพื่อพิมพ์เอกสารในรูปแบบ A4 ระบบจะซ่อนส่วนปุ่มกดและการตั้งค่า แสดงเฉพาะเนื้อหาเอกสารเท่านั้น

!!! note "เอกสารที่ลงชื่อแล้ว"
    หากมีการกำหนดค่า `Signed_Document_URL` ระบบจะแสดงลิงก์ดาวน์โหลดเอกสารที่ลงชื่อแทนตัวอย่างเอกสาร และปุ่มพิมพ์จะถูกปิดใช้งาน
