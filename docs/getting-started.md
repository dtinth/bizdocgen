# เริ่มใช้งาน

หน้านี้อธิบายวิธีเริ่มใช้ bizdocgen ทั้งแบบใช้งานบน Grist ที่โฮสต์ไว้แล้ว และแบบ self-hosted

## วิธีที่ง่ายที่สุด: ใช้เทมเพลตที่โฮสต์ไว้

1. เปิดเทมเพลตที่ <https://bizdocgen.getgrist.com/cq6sb6WHRMre/bizdocgen-template>
2. ทดลองแก้ไขข้อมูลในเอกสารได้ทันที
3. ถ้าต้องการเก็บงานไว้ใน workspace ของตัวเอง ให้กด **Use This Template** หรือ **Save Copy**

!!! note
    Grist มีโหมดทดลองใช้งานที่ช่วยให้เปิดเทมเพลตและแก้ไขข้อมูลได้โดยไม่ต้องลงชื่อเข้าใช้ก่อน

## ใช้กับ Grist ของตัวเอง

ถ้าคุณมี Grist instance ของตัวเอง ให้ใช้ไฟล์ `template.grist` จาก repository เพื่อนำเข้าเป็นเทมเพลต

1. ดาวน์โหลดไฟล์ `template.grist` จาก repository
2. นำเข้าไฟล์เข้าไปใน Grist instance ของคุณ
3. เปิดเอกสารแล้วเลือกเรคอร์ดที่ต้องการพิมพ์
4. เปิดวิดเจ็ตเพื่อแสดงเอกสารจากข้อมูลของแถวปัจจุบัน

## ประเภทเอกสารที่รองรับ

- `Quotation` → ใบเสนอราคา
- `Invoice` → ใบแจ้งหนี้
- `Receipt` → ใบเสร็จรับเงิน

## ปุ่มและความสามารถหลักในหน้าเอกสาร

- **พิมพ์เอกสาร** — เรียกหน้าต่างพิมพ์ของเบราว์เซอร์
- **คัดลอก JSON** — คัดลอกข้อมูลดิบจาก Grist เพื่อใช้ตรวจสอบหรือ debug
- **Custom CSS Settings** — เปิดแผงสำหรับใส่ CSS เพิ่มเติม

ในโหมด standalone จะมีตัวเลือก **Scenario** เพิ่มเข้ามาเพื่อสลับชุดข้อมูลตัวอย่าง

## กรณีพิเศษที่ควรรู้

### เอกสารถูกลงชื่อแล้ว

ถ้าฟิลด์ `Signed_Document_URL` มีค่า แอปจะไม่แสดงเอกสารฉบับพิมพ์ แต่จะแสดงลิงก์ไปยังไฟล์เอกสารที่ลงชื่อแล้วแทน และปุ่มพิมพ์จะถูกปิดใช้งาน

### PromptPay QR Code

ถ้าใน `Payment_Method.PromptPay` มีค่า แอปจะสร้าง QR Code ให้โดยอัตโนมัติโดยใช้ยอดรวมสุทธิของเอกสาร

### ภาษี

- `Tax = 0.07` แสดง VAT 7%
- `Tax < 0` แสดงเป็นภาษีหัก ณ ที่จ่าย เช่น `-0.03` = หัก ณ ที่จ่าย 3%
- `Tax = 0` หมายถึงไม่มีรายการภาษีเพิ่มเติม
